# Handoff

## Open Loops

Work that is active, unresolved, or worth watching.

## Risky Recent Changes

Code changes, config changes, process changes, or decisions in the last 48 hours that may create follow-up work.

## Partially-Mitigated Issues

Known problems with temporary fixes, waiting on permanent resolution.

## Active Threads

Chats, tickets, documents, or conversations that still need attention.

## Time-Sensitive Items

Deadlines, promised follow-ups, or decisions that need action soon.
